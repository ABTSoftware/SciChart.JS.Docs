# Polar Column Chart

The [PolarColumnRenderableSeries](https://www.scichart.com/documentation/js/v4/typedoc/classes/polarcolumnrenderableseries.html) is a type of renderable series that displays data in a polar column format. It is used to visualize data in a circular layout, where each column represents a data point at a specific angle and radius.

## Polar Radial Columns

The same renderable series can be used as radial columns as well, if we flip the roles of the X and Y axes. This is done by swapping the [polarAxisMode](https://www.scichart.com/documentation/js/v4/typedoc/enums/epolaraxismode.html) property in between the axes.

Usually, the xAxis has the [polarAxisMode](https://www.scichart.com/documentation/js/v4/typedoc/enums/epolaraxismode.html) set to `Angular` and the yAxis has it set to `Radial`. To create a radial column chart, we need to set the xAxis to `Radial` and the yAxis to `Polar`.

```ts file=./BasicRadial/demo.js start=region_A_start end=region_A_end
```
