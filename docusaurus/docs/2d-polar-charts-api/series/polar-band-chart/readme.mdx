## Create a Polar Band Series

To create a Javascript [Polar Band Series](https://www.scichart.com/documentation/js/v4/typedoc/classes/polarbandrenderableseries.html) with SciChart.js, use the following code:

```ts showLineNumbers file=./Basic/demo.js start=region_A_start end=region_A_end
```

This results in the following output:

<ChartPreviewWrapper jsContent={simpleBandChart} />

In the code above:

- 2 Polar Band Series instances are created and added to the `sciChartSurface.renderableSeries` collection.
- We set the stroke, fill properties for when Y1 > Y2 and vice versa (more info over at [PolarBandRenderableSeries](https://www.scichart.com/documentation/js/v4/typedoc/classes/polarbandrenderableseries.html) in TypeDoc).
- We assign a DataSeries - in this case an XyyDataSeries which stores X, Y1, Y2 data.
- Optionally, we can set `interpolateLine: true` to make lines wrap around the axes in a circular fashion.

## Gradient Fills in Band Series

The Polar Band Series chart type supports gradient fills as well as solid color fills.

To use this, you need to set the [PolarBandRenderableSeries.fillLinearGradient](https://www.scichart.com/documentation/js/v4/typedoc/classes/polarbandrenderableseries.html#filllineargradient) & [fillLinearGradientY1](https://www.scichart.com/documentation/js/v4/typedoc/classes/polarbandrenderableseries.html#filllineargradienty1) properties with a [GradientParams](https://www.scichart.com/documentation/js/v4/typedoc/classes/gradientparams.html) object: a type which defines gradients by a number of [gradient stops](https://www.scichart.com/documentation/js/v4/typedoc/classes/gradientparams.html#gradientstops), with a start and end point.

Here's the code example and example:

```ts showLineNumbers file=./GradientFill/demo.js start=region_A_start end=region_A_end
```

<ChartPreviewWrapper jsContent={gradientFillBandChart} />

In the code above:
- We create a [GradientParams](https://www.scichart.com/documentation/js/v4/typedoc/classes/gradientparams.html) object with 2 gradient stops, and set the start and end points of the gradient.
- We set [scaleGradientToYRange](https://www.scichart.com/documentation/js/v4/typedoc/classes/polarbandrenderableseries.html#scalegradienttoyrange) to `true` to make the gradient scale to the Y range of the data for each band segment.

## Using a PaletteProvider with Polar Band Series

By extending [DefaultPaletteProvider](https://www.scichart.com/documentation/js/v4/typedoc/classes/defaultpaletteprovider.html) you can create a custom palette for your Polar Band Series. This is useful if you want to set the color of the band based on `xValue`, `yValue` or `index`.

Here is how to do this:

```ts showLineNumbers file=./PaletteProvider/demo.js start=region_A_start end=region_A_end
```

<ChartPreviewWrapper jsContent={PaletteProviderExample} />

In the code above:
- We create a class that extends [DefaultPaletteProvider](https://www.scichart.com/documentation/js/v4/typedoc/classes/defaultpaletteprovider.html) to override [overrideFillArgb](https://www.scichart.com/documentation/js/v4/typedoc/classes/defaultpaletteprovider.html#overridefillargb) and [overrideStrokeArgb](https://www.scichart.com/documentation/js/v4/typedoc/classes/defaultpaletteprovider.html#overridestrokeargb) methods by `xValue`.
