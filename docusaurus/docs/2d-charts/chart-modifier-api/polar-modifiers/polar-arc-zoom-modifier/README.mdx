# PolarArcZoomModifier

import ChartPreviewWrapper from "@site/src/components/ChartPreviewWrapper";

The [PolarArcZoomModifier](https://www.scichart.com/documentation/js/v4/typedoc/classes/polararczoommodifier.html) is a modifier that allows users to zoom in on a specific arc of a polar chart. It provides a visual feedback mechanism for selecting the area to zoom in on, enhancing the user experience by allowing for focused analysis of specific data ranges.
