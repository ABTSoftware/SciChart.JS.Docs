# The Polar Gauge Chart Type

The Gauge chart is used to visualize data in a polar coordinate system, where each point is represented by an angle and a radius.

In SciChart, it is mainly created by using the [PolarColumnRenderableSeries](https://www.scichart.com/documentation/js/v4/typedoc/classes/polarcolumnrenderableseries.html), [PolarStackedColumnRenderableSeries](https://www.scichart.com/documentation/js/v4/typedoc/classes/polarstackedcolumnrenderableseries.html) or [PolarArcAnnotation](https://www.scichart.com/documentation/js/v4/typedoc/classes/polararcannotation.html) classes to make the circular bar / gauge. Both are good alternatives to create a gauge chart and have different use cases.

## Create a Gauge Chart using [PolarArcAnnotation](https://www.scichart.com/documentation/js/v4/typedoc/classes/polararcannotation.html)

To create a Javascript `Polar Gauge Chart` using [PolarArcAnnotation](https://www.scichart.com/documentation/js/v4/typedoc/classes/polararcannotation.html) with SciChart.js, use the following code:

```ts showLineNumbers file=./UsingArc/demo.js start=region_A_start end=region_A_end
```

This results in the following output:

<ChartPreviewWrapper jsContent={usingArc} />

In the code above:
- We Prepare the logic for the gauge chart, total angle and its range.
- We configure the polar axes, and use `isVisible: false` to hide gridlines, ticks and labels.
- (Optional) Add a background gray [PolarArcAnnotation](https://www.scichart.com/documentation/js/v4/typedoc/classes/polararcannotation.html) to show the full potential of the gauge.
- Add the main [PolarArcAnnotation](https://www.scichart.com/documentation/js/v4/typedoc/classes/polararcannotation.html) to represent the current value.
